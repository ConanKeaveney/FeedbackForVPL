The included files included in this folder may be installed onto Moodle by doing the following:

- Navigate to the Moodle folder and copy the vplgrade contents into moodle/mod/vpl/forms/
- Navigate to the Moodle folder and copy the moodlegrade contents into /moodle/grade/report/
- Adjust the DBconnect.php files setting to match the current database


The email functionality should now be usable on Moodle by navigating to a course and viewing the grades->grade analysis.
NOTE: these changes are not visibe when logged in as a student.
